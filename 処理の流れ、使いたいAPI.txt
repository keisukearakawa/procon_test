画像に含まれる単語を検出して、意味や読みを調べてくれるアプリケーション

スマートフォンでの辞書アプリは、読みを入力するのが多く、
読みも分からない時には手書きなどを利用するが、あんまり認識してくれないし、
画数が多いと、書き直しなどの手間があり、めんどくさい。

そこで、カメラで調べたい文字を撮り、読みや意味が分かれば簡単に調べることが出来ると思った。
手早く、誰でもすぐに読みがわかっちゃう！

～処理の流れ～
　①アプリケーションを起動すると、カメラが起動する。
　②撮りたい文字を撮る。
　③写真から文字を検出する　←docomoの文字認識APIを使用。
　④複数文字がある場合は、リストを表示し、調べたい文字をタップしてもらう。
　⑤調べたい文字を辞書APIに入れ、読みと意味を返してもらう。
　⑥返してもらった意味や読みを表示させる。
　

　対応させる文字
　漢字　熟語・四字熟語（まずはこれを対応させる）
　
　英単語（できれば）
　呼び上げ（できれば）
　
　環境：Microsoft Visual Studio 2010 Express for Windows Phon
　      C# XML
